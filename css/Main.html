Centering Block Elements

How it works

Block elements (div, section, main, article…) naturally occupy 100% of available width. To center one horizontally:

  1. Give it a max-width (so it doesn't span the full viewport)
  2. Set margin-left: auto and margin-right: auto
/* ✅ Responsive centering */
main {
  max-width:    960px;   /* shrinks on small screens */
  margin-left:  auto;
  margin-right: auto;

  /* shorthand version: */
  margin: 20px auto;
}

/* ❌ Avoid fixed width — breaks small screens */
main {
  width: 960px;   /* stays 960px even on a 400px phone */
}

CSS Transitions

Syntax

A transition makes a style change happen smoothly over time instead of instantly. Two properties are required; two are optional. 

/* Longhand */
section {
  transition-property:        border-color;  /* required */
  transition-duration:        0.6s;          /* required */
  transition-timing-function: ease;          /* optional */
  transition-delay:           0s;            /* optional */
}

/* Shorthand ← preferred */
section {
  transition: border-color 0.6s ease;
}

/* Multiple properties */
button {
  transition:
    background-color 0.5s ease,
    transform        0.3s ease,
    letter-spacing   0.3s linear;
}

The timing function controls how fast a transition moves at each moment of its duration — not how long it takes overall.

/* Timing options */
ease         /* slow → fast → slow  (default)     */
linear       /* same speed throughout               */
ease-in      /* starts slow, ends fast              */
ease-out     /* starts fast, ends slow              */
ease-in-out  /* slow start + slow end               */

CSS Transforms

Hover each card to see the transform

Transforms move or reshape an element. Combine with transition to make changes smooth. Without a transition the change is instant.

color
scale(1.5)
rotate(180deg)
translateY(-20px)
skewX(-20deg)
multi-transform 
/* Always put transition on the BASE selector, not :hover */
div {
  transition: transform 1s ease;
}

div:hover {
  transform: scale(1.5);
  /* or: rotate(180deg)      */
  /* or: translateY(-20px)   */
  /* or: skewX(-20deg)       */
  /* combine multiple:       */
  transform: rotate(45deg) scale(1.2);
}

Pseudo-class :hover

State-based styling

The :hover pseudo-class activates when the pointer moves over an element. Any CSS property change inside :hover is eligible for a transition — the browser animates from the old value to the new value. 

section {
  border-left: 6px solid coral;
  transition: border-color 0.6s ease;  /* defined HERE */
}

section:hover {
  border-left-color: cyan;               /* change triggered here */
}

Other pseudo-classes that trigger transitions: :focus (form inputs), :active (mouse-down), :checked (checkbox / radio).

Quick Reference

max-width vs width
width: 960px;      /* fixed — never shrinks, breaks phones  */
max-width: 960px;  /* responsive — shrinks below 960px      */
text-align vs margin auto
text-align: center;  /* centers TEXT inside the element  */
margin: 0 auto;      /* centers the ELEMENT itself       */
Transition shorthand order
transition: [property]  [duration]  [timing-fn]  [delay];
/*           required    required    optional     optional  */

/* Example */
transition: border-color 2s ease;
