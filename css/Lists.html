About the Game

Grand Theft Auto VI is the highly anticipated next entry in Rockstar Games' open-world action-adventure series. Set in the fictional state of Leonida, it features the series' first playable female protagonist.

GTA 6

GTA 6 is set to redefine open-world gaming with a massive map, improved AI, and a gripping dual-protagonist narrative. The game is expected to push the boundaries of modern hardware.

New Features

Gameplay Features

The following new gameplay mechanics have been confirmed:

  • Dynamic weather system affecting NPC behaviour
  • Seamless online integration from day one
  • Expanded vehicle and weapon customisation
  • Fully destructible environments
  • Realistic economy and property system
